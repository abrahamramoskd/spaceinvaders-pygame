# 👾 Space Invaders en Python

Recreación del clásico arcade en un **solo archivo Python** usando `pygame`.
Sprites dibujados en código puro, sin imágenes externas.

## ✨ Características

- 3 tipos de aliens con animación de 2 poses (55 aliens por oleada)
- Aliens se aceleran al quedar menos en pantalla y entre oleadas
- 4 bunkers destructibles bloque a bloque
- Alien misterioso con puntos aleatorios (50/100/150/300)
- Sistema de 3 vidas con parpadeo de invencibilidad
- Explosiones con partículas y puntos flotantes
- Fondo de estrellas con efecto parallax
- Código comentado línea a línea con fines didácticos

## 🚀 Instalación

```bash
pip install pygame
python space_invaders.py
```

## 🕹️ Controles

| Tecla     | Acción              |
|-----------|---------------------|
| `← →`    | Mover nave          |
| `ESPACIO` | Disparar            |
| `P`       | Pausar / reanudar   |
| `R`       | Reiniciar partida   |
| `ESC`     | Salir               |
